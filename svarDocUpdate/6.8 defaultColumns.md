# 6.8 defaultColumns

# **Description**

An array of objects with default configuration parameters for the Gantt columns in the grid area

# **Default config**

```jsx
defaultColumns = [
   { id: "text", header: "Task name", flexgrow: 1, sort: true },
   { id: "start", header: "Start date", align: "center", sort: true },
   {
      id: "duration",
      header: "Duration",
      width: 100,
      align: "center",
      sort: true,
   },
   {
      id: "add-task",
      header: "Add task",
      width: 50,
      align: "center",
      sort: false,
      resize: false,
   },
];

```
