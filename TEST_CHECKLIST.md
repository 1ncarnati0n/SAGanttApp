## 🔧 문제 발생 시

### ❌ Dashboard가 비어있음
**Mock 모드**: 
- LocalStorage 초기화: `localStorage.clear()` (브라우저 콘솔)
- 페이지 새로고침

**Supabase 모드**: 
- "샘플 프로젝트 생성" 버튼 클릭
- 또는 Supabase 스키마 재실행

### ❌ "샘플 프로젝트 생성" 실패
1. 터미널에서 에러 확인
2. Supabase 스키마 재실행 필요
3. 또는 Mock 모드로 전환

### ❌ Gantt 차트가 열리지 않음
1. Task 데이터 확인 (Supabase Table Editor)
2. 브라우저 콘솔에서 에러 확인
3. 페이지 새로고침

### ❌ "Mock 모드" 배지가 사라지지 않음
1. `.env.local` 파일 확인
2. Supabase 환경 변수 설정 확인
3. 서버 재시작
